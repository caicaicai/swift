# Laravel 4.1 中文文档

Laravel 4.1 中文文档是在 4.0 中文文档的基础上演变而来的，对某些新增的内容进行了修订。

## 如何参与翻译

此文档会定期与英文文档进行同步，根据差异修订中文文档，因此，如果你看到的某篇文档已经大部分被翻译，但是某些内容保留了英文原文，那就是新修订的部分，你可以随时clone本文档，补足中文翻译后直接提交Pull Request即可。

所有为本文档贡献的朋友都将展示在“贡献者名单”。
